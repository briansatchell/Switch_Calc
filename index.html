About this app

Instructions for using the app:

  • Use the keys "W" (up), "A" (previous), "S" (down), "D" (next)
    to navigate the calculator and "F" to select the highlighted button
  • To simulate a 5-button switch control use all 5 of the above keys
  • To simulate a 3 button switch control use "A" (next) "D" (previous) and "F" (select)
